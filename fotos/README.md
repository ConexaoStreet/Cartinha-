# Fotos

Esta pasta fica versionada apenas com este arquivo de texto. Não coloque fotos reais no repositório se você precisa criar PR por uma ferramenta que não aceita arquivos binários.

Depois de publicar ou baixar o projeto, adicione manualmente suas imagens nesta pasta com estes nomes para a galeria funcionar:

- `foto1.jpg`
- `foto2.jpg`
- `foto3.jpg`
- `foto4.jpg`
- `foto5.jpg`

Os caminhos já estão configurados no código como `fotos/foto1.jpg` até `fotos/foto5.jpg`. Use imagens otimizadas, de preferência com até 1 MB cada, para o site carregar rápido no celular.
